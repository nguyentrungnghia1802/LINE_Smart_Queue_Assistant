# Documentation Archive

This directory contains historical artifacts retained for project history only. Files here may describe superseded requirements, demo data, architecture, release state, or unverified assumptions.

Do not use archive files as implementation truth. Start with `README.md`, `AGENTS.md`, and `docs/00_PROJECT_CONTEXT.md`; consult the archive only when a task explicitly requires historical investigation.
