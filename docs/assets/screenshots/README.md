# Screenshots

Place README screenshots in this directory.

Recommended files:

- `customer-liff-home.png`
- `customer-booking.png`
- `customer-ticket.png`
- `staff-dashboard.png`
- `manager-dashboard.png`
- `manager-qr.png`

Use optimized PNG or WebP files and avoid committing images that contain secrets,
real customer data, LINE channel tokens, payment identifiers, or private location
details.
